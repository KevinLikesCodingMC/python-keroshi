## ✨ Features

✨ Convert ZIP with WebP to a single PDF!

```python
from keroshi import jm2pdf

jm2pdf("1122530.zip")
```

## 🚀 Installation

Download the wheel from releases!

```bash
pip install keroshi-0.0.1-py3-none-any.whl
```
